        MFP Control Center
        Центр управления МФУ HP LaserJet M1536dnf
═══════════════════════════════════════════════════════════════

УСТАНОВКА (ОЧЕНЬ ПРОСТО!):
--------------------------
1. Подключите принтер HP LaserJet M1536dnf по USB
2. Дважды кликните на файл "УСТАНОВИТЬ.bat"
3. Если Windows спросит - нажмите "Да"
4. Если нет Visual Studio - установщик АВТОМАТИЧЕСКИ
   скачает и установит бесплатные Build Tools (~5-15 мин)
5. Дождитесь завершения установки
6. Готово! Ярлык появится на рабочем столе


ЧТО ПРОИСХОДИТ ПРИ УСТАНОВКЕ:
-----------------------------
1. Проверка .NET Framework 4.8 (обычно уже есть в Windows 10)
2. Поиск Visual Studio / Build Tools
3. Если не найдено - АВТОМАТИЧЕСКАЯ загрузка и установка
   Visual Studio Build Tools 2022 (бесплатно, ~2-3 ГБ)
4. Сборка программы из исходного кода
5. Копирование в C:\Program Files\MFP Control Center
6. Создание ярлыка на рабочем столе


ТРЕБОВАНИЯ:
-----------
- Windows 10 64-bit
- Интернет (для скачивания Build Tools, если их нет)
- ~3 ГБ свободного места (для Build Tools)


ФУНКЦИИ ПРОГРАММЫ:
------------------

  ПЕЧАТЬ
  - Односторонняя и двусторонняя печать
  - Выборочная печать страниц (1-5, 1,3,5-7)
  - Настройка качества и размера бумаги
  - Поддержка PDF и изображений

  СКАНЕР
  - Предпросмотр перед сканированием
  - Сканирование со стекла и из автоподатчика (ADF)
  - Выбор разрешения: 75-1200 DPI
  - Сохранение в PDF, JPEG, PNG, TIFF
  - Многостраничное сканирование в один PDF

  КОПИР (КСЕРОКС)
  - Мгновенная копия (сканирует и сразу печатает)
  - Отложенная копия (сканирует всё, потом печатает)
  - ID-копия (обе стороны документа на одном листе)
  - Настройка масштаба (25-400%)
  - Настройка яркости и контрастности


УДАЛЕНИЕ ПРОГРАММЫ:
-------------------
Запустите файл в папке установки:
C:\Program Files\MFP Control Center\Удалить программу.bat


ПРОБЛЕМЫ И РЕШЕНИЯ:
-------------------

Проблема: "Принтер не найден"
Решение: Проверьте USB-подключение и установите драйверы HP

Проблема: "Сканер не найден"
Решение: Установите драйвер сканера с официального сайта HP

Проблема: Ошибка при установке Build Tools
Решение: Проверьте интернет-соединение, попробуйте снова

Проблема: "Требуется перезагрузка"
Решение: Перезагрузите компьютер и запустите установщик снова


КОНТАКТЫ:
---------
При возникновении проблем обратитесь к разработчику.
